# Hands-on-AI-Tutorial

Welcome to the **Hands-on AI Tutorial**! This is a fully open-source project designed to teach you how to build, train, and use Artificial Intelligence models from scratch. 

Whether you are a beginner or looking to solidify your understanding of AI, this tutorial will guide you through the process step-by-step. The core philosophy of this project is to not only show you *how* to write the code to load and train models, but also to help you understand the *data* and the *mathematics* behind the algorithms.

## Project Structure

This repository is divided into two main projects, each focusing on a core area of Artificial Intelligence:

### 1. Machine Learning Fundamentals (`ml_fundamentals`)
This section covers the foundational algorithms of machine learning. It is divided into three chapters:
*   **Chapter 1: Regression** - Learn how to predict continuous values (e.g., predicting house prices).
*   **Chapter 2: Classification** - Learn how to categorize data into distinct classes (e.g., spam detection).
*   **Chapter 3: Clustering** - Learn how to group unlabeled data based on similarities (e.g., customer segmentation).

### 2. Deep Learning (`deep_learning`)
This section dives into neural networks and more advanced AI architectures. It is divided into two chapters:
*   **Chapter 1: Natural Language Processing (NLP)** - Learn how to make computers understand, interpret, and generate human language.
*   **Chapter 2: Computer Vision (CV)** - Learn how to enable computers to extract meaningful information from digital images and videos.

## What You Will Learn
*   **Data Handling:** How to load, clean, and preprocess datasets for training.
*   **Model Training:** How to build, compile, and train models using popular frameworks.
*   **The Math:** An intuitive breakdown of the mathematical concepts that power these models.
*   **Evaluation:** How to test your models and measure their performance.
