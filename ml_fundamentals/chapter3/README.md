# Chapter 3: Clustering

## Objective
The main objective of this chapter is to explore Clustering, a core *unsupervised* learning technique. Unlike regression and classification, clustering involves finding hidden patterns, structures, or groupings in data that does not have pre-labeled outcomes (e.g., segmenting customers based on purchasing behavior).

In this chapter, you will learn:
* The math and intuition behind clustering algorithms (like K-Means).
* How to work with and scale unlabeled datasets.
* How to train clustering models and determine the optimal number of clusters.
* Methods to evaluate the quality of the clusters formed without having true labels.
