# Chapter 2: Classification

## Objective
The main objective of this chapter is to dive into Classification algorithms. Classification is a supervised learning approach used when the target variable is categorical, meaning we want to assign an input to one of a few distinct classes (e.g., identifying whether an email is "spam" or "not spam").

In this chapter, you will learn:
* The underlying mathematics of classification algorithms (such as Logistic Regression).
* Techniques for handling and preprocessing categorical data.
* How to train, test, and fine-tune classification models.
* Important classification evaluation metrics (Accuracy, Precision, Recall, F1-Score).
